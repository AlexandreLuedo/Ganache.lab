* Ganache.lab
Le premier équilibreur de ganache multiplateforme !

#+begin_quote
[!NOTE]
Ceci est mon premier projet, il sert de support à mon apprentissage de Dart & Flutter. L'architecture est donc amenée à évoluer régulièrement.
#+end_quote

** Fonctionnalités
L'application Ganache.lab permet l'équilibrage précis de ganaches via deux approches principales :

*** 1. Calcul du poids cible (Mode Géométrique)
L'utilisateur définit ses contraintes physiques (moulage, cadrage ou poids libre). L'application calcule alors le poids total nécessaire pour remplir le volume demandé.

*** 2. Équilibrage de la recette (Mode Algorithmique)
C'est le cœur du MVP. L'utilisateur saisit ses ingrédients un par un. L'application calcule en temps réel :
- **Humidité (%)** : Taux d'eau total dans la recette.
- **Texture / Matière Grasse (%)** : Équilibre des graisses (beurre de cacao + gras lactique).
- **Pouvoir Sucrant (PS/POD)** : Intensité sucrée de la ganache.
- **Extrait Sec Total (EST)** : Pourcentage de matières solides.

** Architecture Technique (v2.0)

*** Modèle d'Ingrédients Unifié
Pour permettre des calculs précis, tous les ingrédients implémentent désormais l'interface ~Ingredient~ dans ~lib/services/calculation.dart~ :
#+begin_src dart
abstract class Ingredient {
  String get name;
  double get waterContent;  // % (0.0 to 1.0)
  double get fatContent;    // % (0.0 to 1.0)
  double get sugarContent;  // % (0.0 to 1.0)
  double get solidsContent; // % (0.0 to 1.0)
}
#+end_src

*** Gestion d'état (Provider)
L'application utilise le pattern **Observer** via ~ChangeNotifier~ :
- ~RecipeNotifier~ : Gère la liste des ingrédients sélectionnés et leurs poids respectifs.
- ~TotalModel~ : Gère les calculs géométriques (cadres/moules).
- ~ApplicationModel~ : Gère l'état de la navigation entre les modes.

*** Conventions de code
- *Langue :* Commentaires et variables en **Anglais** (standard industriel).
- *Pourcentages :* Toujours exprimés entre ~0.0~ et ~1.0~ pour simplifier les calculs (ex: 15% = 0.15).

** Documentation
La documentation technique est intégrée via le tag ~// DOC~.

*** Glossaire technique
- **POD (Point of Overrun)** : Pouvoir Sucrant. Vient du terme italien *Potere Dolcificante*.
- **EST** : Extrait Sec Total (Total Solids).
- **MSNF** : Milk Solids-Not-Fat (Extrait Sec Dégraissé du Lait).
