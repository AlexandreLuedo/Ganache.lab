# Ganache.lab

Le premier équilibreur de ganache multiplatformes !

> ⚠️ Ceci est mon premier projet, il me permet d&rsquo;apprendre Dart & Flutter !


## Architecture


### Commentaires

Les commentaires sont rédigés en français (pour la plus part) sinon je changerai les commentaires rédigés en anglais vers le français.


### Variables

Le nom des variables est en anglais, meêmes si certaines (pour l&rsquo;instant) possèdent des parties anglais comme `numberCadres` utilisée pour stocker le nombre de cadre que l&rsquo;utilisateur renseigne.
Le nom des variable est réellement courts, peux être à terme je serai plus claire pour leurs rédcation.


## Documentation

Les commentaires sont rédigés en Français. Les pages de documentation sont des commentaires contenants `// DOC` en préambule.


### Ex

    // DOC
    // La variable ganache est chocolatée car elle contient la variable chocolat.


### Glossaire

Le glossaire est une documentation destinée aux utilisateurs. Elle permet d&rsquo;expliquer à l&rsquo;utilisateur comment fonctionne la ganache. Elle à été exprès rédigée en markdown pour une modification qui puisse entre faîte par chacuns (à voir comment je ferai pour rendre la modification accessible).
