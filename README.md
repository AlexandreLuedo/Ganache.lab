# Ganache.lab

Le premier équilibreur de ganache multiplatformes !

> ⚠️ Ceci est mon premier projet, il me permet d&rsquo;apprendre Dart & Flutter !


## Architecture


### Commentaires

Les commentaires sont rédigés en français (pour la plus part) sinon je changerai les commentaires rédigés en anglais vers le français.


### Variables

Le nom des variables est en anglais, mêmes si certaines (pour l&rsquo;instant) possèdent des parties anglais comme `numberCadres` utilisée pour stocker le nombre de cadre que l&rsquo;utilisateur renseigne.
Le nom des variable est réellement courts, peux être à terme je serai plus claire pour leurs rédcation.


### Architecture de code

-   `GanacheApp()` est le début du programme.
-   `navigation.dart` la &ldquo;home page&rdquo;.

-   MIGRATION

    J&rsquo;essaie de donner une organisation plus logique à mon projet. Certaines parties du code ne sont pour l&rsquo;instant même pas utilisés. La transition vers une architecture clean se fera en plusieurs commits qui ne seront pas spécialements dédiés à l&rsquo;achitecture du projet.


## Documentation

Les commentaires sont rédigés en Français. Les pages de documentation sont des commentaires contenants `// DOC` en préambule.
**Ex**

    // DOC
    // La valeur var ganache = 1; permet un calcul de celle-ci.


### Glossaire

Le glossaire est une documentation destinée aux utilisateurs. Elle permet d&rsquo;expliquer à ceux-ci comment fonctionne la ganache. Cette documentation a été exprès rédigée en markdown pour une modification plus accessible (à voir comment je ferai pour rendre la modification possible pour les utilisateurs).
Le glossaire est disponible dans `assets/documentations/doc.md`
